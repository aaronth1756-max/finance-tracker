## ⚙️ Installation

Clone the repository

```
git clone YOUR_GITHUB_REPOSITORY
```

Install dependencies

```
composer install
```

Create a MySQL database

```
finance_tracker
```

Import the SQL file.

Update the database configuration.

Run

```
php -S localhost:8000
```

or use XAMPP.
